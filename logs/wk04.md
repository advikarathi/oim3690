## What I learned

### Flexbox
- `justify-content` controls horizontal alignment, `align-items` controls vertical
- Good for one-directional layouts like navbars or a row of cards

### Grid
- `grid-column` and `grid-row` let you place items precisely in a two-axis layout
- Better when you need rows AND columns at the same time

### Responsive Design
- The viewport meta tag is what actually makes mobile scaling work — without it
  the browser just shrinks the desktop version
- DevTools phone view is way faster than pulling out your phone every time

### HTML Forms
- The `action` attribute defines where form data gets sent; `method` defines how
  (GET appends to URL, POST sends in the request body)
- Each input needs a `name` attribute — that's the key the server receives
- Labels should use `for` matching the input's `id` for accessibility
- HTML5 validation (e.g. `required`, `type="email"`) catches errors before
  the form even submits
